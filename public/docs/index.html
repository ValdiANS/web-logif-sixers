Mendapatkan data makanan pokok

Endpoint: https://sixers.netlify.app/.netlify/functions/makanan_pokok
Request Method : GET

Mendapatkan data makanan pokok yang telah dikalkulasi dengan metode SMART

Endpoint: https://sixers.netlify.app/.netlify/functions/makanan_pokok
URL Parameter :

  • isSmart: boolean
  • kriteria: bulking/diet/jagaBB
Request Method : GET
Misalnya ketika ingin mendapatkan data makanan pokok yang telah dikalkulasi dengan metode SMART untuk menjaga berat badan, maka URL parameternya bernilai seperti berikut :
  • isSmart=true
  • kriteria=jagaBB
Sehingga URL endpointnya adalah :
https://sixers.netlify.app/.netlify/functions/makanan_pokok?isSmart=true&kriteria=bulking
